# 🎓 منصة أكاديميا - المنصة التعليمية

منصة تعليمية متقدمة لتعليم البرمجة وتطوير الويب من الصفر إلى الاحتراف.

## ✨ المميزات

- 🎨 تصميم حديث وجذاب
- 📱 متجاوب مع جميع الأجهزة (Responsive Design)
- ⚡ سرعة عالية في التحميل
- 🎯 واجهة سهلة الاستخدام
- 💬 نموذج تواصل فعال

## 📚 الدورات المتاحة

1. **HTML & CSS** - تعلم أساسيات بناء صفحات الويب
2. **JavaScript** - احتراف لغة البرمجة الأساسية للويب
3. **Web Design** - تصميم واجهات مستخدم احترافية
4. **React.js** - بناء تطبيقات ويب حديثة وديناميكية

## 🚀 خطوات النشر على GitHub Pages

### الطريقة 1: عبر GitHub Pages (الأسهل والأسرع)

1. **اذهب إلى إعدادات المستودع:**
   - انقر على `Settings` في الأعلى

2. **فعّل GitHub Pages:**
   - اذهب إلى قسم `Pages` من الجانب الأيسر
   - في `Source`، اختر `Deploy from a branch`
   - اختر الفرع `main` والمجلد `/ (root)`
   - انقر على `Save`

3. **انتظر دقائق:**
   - ستظهر رسالة بـ URL الموقع الخاص بك
   - الموقع سيكون متاحاً على: `https://yourusername.github.io/repo-name`

## 🎯 خطوات الاستخدام

1. **استنساخ المستودع:**
```bash
git clone https://github.com/Mozaky73/https-academia-platform-91995.web.app-.git
cd https-academia-platform-91995.web.app-
```

2. **فتح الموقع محلياً:**
   - افتح ملف `index.html` بمتصفحك

3. **تطوير الموقع:**
   - عدّل الملفات حسب احتياجاتك
   - أضف محتوى جديد للدورات

4. **رفع التغييرات:**
```bash

git commit -m "تحديث المحتوى"
git push origin main
```
